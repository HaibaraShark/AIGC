# AI 国风文创生成工具

## 项目概述

AI 国风文创生成工具是一个基于深度学习技术的创意生成系统，旨在结合人工智能与中国传统文化，为文创产品设计提供全新的创作工具。该工具通过扩散模型生成基础图像，结合风格迁移、图像优化和纹样生成等技术，实现从文本到国风文创产品的完整流程。

## 技术架构

### 核心技术栈
- **扩散模型（Diffusion Model）**：基于U-Net架构实现，用于生成基础图像
- **风格迁移**：基于VGG19特征提取网络，实现国风风格的转换
- **图像优化**：包含超分、边缘增强和修复功能
- **图像分割**：用于提取图像中的元素
- **纹样生成**：生成重复的国风纹样，适用于包装、服饰等

## 模型流程

### 1. 图像生成流程

1. **初始化扩散模型**：加载U-Net网络，设置扩散参数
2. **逆向采样**：从随机噪声开始，逐步去噪生成图像
3. **风格迁移**：将生成的基础图像转换为国风风格
4. **图像优化**：对图像进行超分、边缘增强等处理
5. **分割与纹样生成**：提取图像元素，生成重复纹样

### 2. 核心模块详解

#### 扩散模型
- **U-Net架构**：编码器-解码器结构，用于捕获图像的多尺度特征
- **扩散过程**：前向扩散添加噪声，逆向扩散去噪生成图像
- **采样策略**：使用逐步去噪的方式生成高质量图像

#### 风格迁移
- **VGG19特征提取**：提取图像的内容特征和风格特征
- **损失计算**：内容损失保持图像内容，风格损失转移风格特征
- **优化过程**：通过迭代优化生成风格化图像

#### 图像优化
- **超分辨率**：提高图像分辨率，增强细节
- **边缘增强**：突出图像边缘，提升视觉效果
- **图像修复**：修复图像中的缺陷和噪声

#### 纹样生成
- **图像分割**：提取图像中的元素和轮廓
- **纹样设计**：生成几何、花卉和中国传统纹样
- **无缝拼接**：制作可重复使用的无缝纹样

## 使用方法

### 环境搭建

1. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```

2. 运行主脚本
   ```bash
   python main.py
   ```

3. 查看生成结果
   - `generated_image.png`：生成的基础图像
   - `optimized_image.png`：优化后的图像
   - `pattern.png`：生成的纹样
   - `segment_mask.png`：分割掩码

### 示例代码

```python
from main import AIGCApplication

# 初始化应用
app = AIGCApplication()

# 生成国风图像
generated_image = app.generate_image(style='chinese')
generated_image.save('output/generated.png')

# 优化图像
optimized_image = app.optimize_image(generated_image)
optimized_image.save('output/optimized.png')

# 生成纹样
pattern = app.generate_pattern(style='chinese')
pattern.save('output/pattern.png')

# 分割图像
mask = app.segment_image(generated_image)
mask.save('output/mask.png')
```
